# Proyectofinal

Proyecto de Parking para proyecto final de grado superior de Administración de Sistemas Informáticos en Red.

Se usa Terraform para el despliegue automático en AWS, el cual crea una base de datos con una configuración ya establecida y un servidor web donde se despliega la aplicación creada en React+JS y listo para accecer a la web desde el dominio configurado.
